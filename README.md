# matankify-v2
Matankify API (client-side API for Cubelify)

# Requirements
[Pug API Key](https://discord.gg/xb9R655xJ2): /api-key

[Urchin API Key](https://discord.gg/zmzGx3Hr8M): /dashboard

# How to use
Set the key and type .help

# Config
Path: ``%appdata%/.matankify/config``

Set the content to:
```
Pug=YOUR_PUG_API_KEY
Urchin=YOUR_URCHIN_API_KEY
```
(Make sure to replace them to your valid keys)
Then restart the app
